## Il Metodo (scheletro universale)

1. **Chi / Cosa** — sostantivi base + pronomi personali
2. **Dove / Quando** — avverbi di luogo e tempo
3. **Come / Perché** — connettivi logici, la "colla" della lingua
4. **Sintassi** — ordine delle parole, struttura del verbo
5. **Memorizzazione massiva** — esposizione, flashcard, immersione

> Dopo lo scheletro è tutta ripetizione + esposizione. Film, musica, parlare con nativi.
